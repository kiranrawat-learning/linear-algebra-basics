Chapter 1

Vectors

A vector is an arrow with two properties: direction and length. The point (2, 3) means "2 steps right, 3 steps up." Every vector starts from the origin.

3.0

Key insight: Two special vectors — e₁ = (1, 0) and e₂ = (0, 1) — are called basis vectors. Every other vector is just a combination of these two. For example: (2, 3) = 2·e₁ + 3·e₂
Chapter 2

Matrices

A matrix is a grid of numbers that encodes a transformation. A 2×2 matrix has 2 rows and 2 columns. The columns tell you where the basis vectors go.

Matrix A = [[a, b], [c, d]]

Column 1 = (a, c) → this is where e₁ goes
Column 2 = (b, d) → this is where e₂ goes

A × (x, y) = (ax + by, cx + dy)
The big idea: Once you know where e₁ and e₂ go — every other vector follows automatically. That's the whole magic of matrices!
Matrix valueEffect on e₁Effect on e₂
aStretches e₁ in x-directionNo effect
bNo effectTilts e₂ toward x-axis (x-shear)
cTilts e₁ toward y-axis (y-shear)No effect
dNo effectStretches e₂ in y-direction
Chapter 3

Linear Transformation

A linear transformation takes every vector and moves it to a new place — without bending lines or moving the origin. The matrix is the "recipe" for this movement.

Chapter 4

Shear

Shear is when one coordinate starts depending on the other. Think of a stack of cards — push them sideways and each card shifts based on its height.

0.0
Move the slider to see shear in action.
X-Shear:
x' = x + b·y
y' = y (unchanged)

Matrix = [[1, b], [0, 1]]
e₁ stays, e₂ tilts toward x
Y-Shear:
x' = x (unchanged)
y' = c·x + y

Matrix = [[1, 0], [c, 1]]
e₂ stays, e₁ tilts toward y

Chapter 7

Inverse Matrix

The inverse matrix undoes a transformation. If T moved e₁ to (2,0), then T⁻¹ moves (2,0) back to e₁. It's the "undo button" for transformations.

2.0
1.0
0.0
1.0
...
If A = [[a, b], [c, d]] and det = ad - bc

A⁻¹ = (1/det) × [[ d, -b], [-c, a]]

Three steps:
1. Swap a and d (diagonal)
2. Negate b and c
3. Multiply everything by 1/det
Why do we need inverse? To solve equations! Ax = b → x = A⁻¹·b. Also used in: decryption, 3D rendering (undo camera rotation), and machine learning (going back to original space).
When does inverse NOT exist? When det = 0. The space collapsed to a line — you can't "un-collapse" it. Information was destroyed forever.
Chapter 8

Non-Square Matrices

Square matrices keep dimensions the same (2D→2D). Non-square matrices change dimensions — they either compress or expand the space.

2×3 matrix (Fat):
Takes a 3D vector in
Gives a 2D vector out

3D → 2D (compress)
One dimension is lost
3×2 matrix (Tall):
Takes a 2D vector in
Gives a 3D vector out

2D → 3D (expand)
One dimension is added
Inverse? Regular inverse doesn't exist for non-square matrices. But there's a Pseudo-inverse (A⁺) that gives the "best possible" approximation. Used in machine learning (neural networks, embeddings) and statistics (least squares).
