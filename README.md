# Linear Transformations — A Visual Guide

An interactive visual guide to understanding Linear Algebra from scratch.

## 🔗 Live Site
👉 https://yourusername.github.io/linear-algebra

## 📚 What's Inside
- **Vectors** — what they are, basis vectors, length
- **Matrices** — how columns map to basis vectors
- **Linear Transformation** — step by step with visuals
- **Shear** — X-shear and Y-shear explained
- **Scale & Rotation** — with cos/sin intuition
- **Determinant** — area scaling, collapse, flip
- **Inverse Matrix** — undo button, formula, when it fails
- **Non-Square Matrices** — 2×3 vs 3×2, pseudo-inverse

## 🎯 Who is this for?
Anyone who wants to understand linear transformations visually — 
no heavy math background needed. Built with plain HTML, CSS and Canvas.
